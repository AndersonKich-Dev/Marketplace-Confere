## 🎥📀 Running the web application (Front End)

```bash
# Clone o projeto.

$ git clone https://github.com/AndersonKich-Dev/Marketplace-Confere.git

```

```bash
# Entre no diretório da aplicação.

$ cd Marketplace-Confere-master

# Execute os seguintes comandos.
$ yarn
$ yarn start

```
